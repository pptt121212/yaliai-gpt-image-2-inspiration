# PPT Styles

Use the live Yali `/ppt/` style set as the PPT style standard. Reference open-source PPT skills only for workflow structure, not as the source of the style list.

## Yali PPT Style Set

- `gradient-glass` - 空间玻璃：类似 Apple Vision OS 的空间玻璃质感，适合高级科技发布会。
- `clean-tech-blue` - 蓝白科技：类似 Stripe / Linear 的蓝白商务风，适合 SaaS、策略汇报和产品方案。
- `china-red-official` - 中国红：庄重正式，红金暖白深灰，对称布局和清晰层级。
- `soft-gradient` - 柔和渐变：浅色渐变、圆角卡片、柔和阴影和轻量玻璃层次。
- `premium-dark` - 深色高级：炭黑、午夜蓝、深灰和少量金属色，适合高级感和舞台感。
- `clean-data` - 洁净数据：浅色背景、细网格、指标卡片和图表面板，适合报告型 PPT。
- `luxury-serif` - 高级衬线：大号衬线标题、克制配色、画册式留白，适合品牌感和正式叙事。
- `organic-paper` - 自然纸感：米白纸张纹理、低饱和色和温和排版，适合温暖可信内容。
- `bold-pop` - 大胆波普：高饱和色块、粗体标题、贴纸感图形和强对比构图。
- `vector-illustration` - 复古矢量插画：黑色描边、友好插画感，适合教育、品牌故事和轻松叙事。
- `playful-kids-illustration` - 童趣插画：明亮柔和配色、圆润图形和手绘插画质感，适合儿童向内容。
- `editorial-mono` - 杂志黑白：编辑部杂志感与克制黑白版式，适合文化、品牌和观点型内容。
- `dark-aurora` - 暗色极光：深色背景和霓虹极光氛围，适合 AI 产品、开发者工具和科技发布。
- `risograph` - 孔版印刷：双色专色、网点颗粒和印刷纹理，适合创意工作室与年轻化表达。
- `japanese-wabi` - 日式侘寂：类似无印良品和原研哉的克制留白，适合生活方式与文化内容。
- `east-asian-ink-luxury` - 东方水墨奢雅：宣纸纹理、水墨山水、雾灰层次和红色印章，适合文化、品牌和深度主题。
- `swiss-grid` - 瑞士网格：包豪斯 / Vignelli 式严谨网格，适合研究报告和严肃信息设计。
- `hand-sketch` - 手绘白板：手绘笔记和白板草图风，适合工作坊、头脑风暴和教学解释。
- `y2k-chrome` - Y2K 镀铬：液态镀铬、贴纸感和流行文化视觉，适合年轻化品牌表达。
- `custom` - 自定义风格：按用户文字描述生成统一视觉方向。

## Selection Rule

Choose the style by topic and audience first, not by visual novelty. If the user does not specify a style, pick the closest Yali `/ppt/` style and state it.

## Style Output

Each style should define:

- color palette
- font mood
- grid and spacing rules
- icon or illustration language
- edge treatment and texture level
- content density expectations

## Deck Consistency

Once a style is chosen, keep it stable across the full deck. Only the cover or section divider may deviate slightly for emphasis.

## Source Of Truth

The source of truth is the style set exposed by `https://www.yaliai.com/ppt/`. If the website style list changes later, update this file to match the live site.
